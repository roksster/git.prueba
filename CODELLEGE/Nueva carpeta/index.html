Marcos de trabajo de Java

La palabra inglesa "FRAMEWORK" (marco de trabajo) define, en terminos generales, un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como referenca para enfrentar nuevos prolemas de indole similar.

En el desarrollo de software, un framework o infraestructura digital, es una estructura conceptual y técnologica de soporte definido, normalmente con artefactos o modulos de software concretos que ouede servir de base para al organización y desarrollo de software. Tipicamente hablando puede incluir soporte de programas, bibliotecas y un lenguaje interpretado, entre otras herramientaas, para así ayudar a desarrolllar y unir componentes de un proyecto.
