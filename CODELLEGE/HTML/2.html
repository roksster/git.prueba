Ejemplo de Applet con Java


Un applet conforme a lo que encontramos en las páginas de Oracle: "es una aplicación hecha en Java que se llama desde un documento HTML y se ejecuta dentro de un navegadir". Es necesario recordar que el navegador debe tener instalado el plugin de la Maquina Virtual de Java (JVM).

Maquina Virtual de Java. Es el programa que se encarga de ejecutar el bytecode generado por el compilador de Java.

Para entender la forma en que se manejan los applets vamos a ver un ejemplo que presenta el clásico HOLA MUNDO:

import java.awt.Graphics;
import java.swing.JApplet {
public void paint(Graphics g) {
super.paint(g);
g.drawString("Hola mundo!!",10,20);
}
}

Como puede observarse la única variable que utiliza ete applet es g que representa un objeto de la case Graphics sobre el cual vamos a dibujar la cadena de texto que resulta de ejecutar el apple.
